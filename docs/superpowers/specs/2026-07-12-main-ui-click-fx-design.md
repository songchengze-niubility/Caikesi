# 主界面点击动效设计

日期：2026-07-12 · 状态：用户已在对话中确认（范围 A + 特效 A 款）

## 目标

主界面当前点击反馈只有瞬时 `setScale(0.94)`，感知接近于无；本设计给主界面全部热区加上可感知的按压动效与点击特效。**范围只限 `MainScreenView`**，各面板暂不动（后续批量迁移另立任务）。

## 效果定义

### 按钮本体反馈（缩放 + 微暗）
- 按下：约 0.08s `quadOut` 缩到基准 ×0.92，同时精灵颜色压暗（乘 ~85% 灰度）。
- 松手：约 0.18s `backOut` 回弹到基准 1.0（带过冲），颜色还原；`onClick` 仍在松手瞬间触发，不等动画播完。
- TOUCH_CANCEL：只还原，不触发点击。
- **穿帮修复**：导航文字与按钮底图是兄弟节点，反馈目标从单精灵扩成「精灵 + 关联文字」一组同步缩放（文字只缩放不变色）。

### 点击特效（纯代码墨圈涟漪）
- 按下瞬间在触点处生成 `Graphics` 圆环：初始缩放 ~0.35 → 0.3s `quadOut` 扩散到 ~1.6，`UIOpacity` 同步淡出到 0 后销毁节点。
- 颜色为墨色描边（`38,36,31`，与 UI 文字同源；2026-07-12 用户验收反馈：初版金色判退，改墨色），半径/线宽/时长/颜色全部提成文件顶部常量供调参。
- 将来 Codex 出墨溅序列帧后，同一触发点直接替换为序列帧播放（B 款），本轮不做。
- **2026-07-12 追加裁决**：Graphics 圆环形状被用户判"太丑"，定为过渡实现；已选定素材路线——2~3 张灰度墨团单帧贴图（RGB 纯白、alpha 承载浓淡、运行时按区域染浓墨/淡墨，随机变体+旋转），任务单 `docs/visual/staging/effects/2026-07-12_click-ink-splat-v01/brief.md`，Codex 出图过审后 Claude 接线，动画沿用现有 tween 与常量。

## 覆盖范围

- 按压反馈：`buildHotZones` 现有全部热区——五导航（含文字）、更换装备（含文字）、金币/宝石栏（含数字）、三个角色头像。不新增可点区域。
- 涟漪：**全屏任意触点**（2026-07-12 用户验收反馈追加）——`root` 挂 TOUCH_START 监听兜底空白处；热区自身 stopPropagation 并各自出涟漪，不双重触发；不拦截冒泡，`BattleEntry` 根节点「点击重开」等祖先监听不受影响。面板打开时触点落在面板上，不出主界面涟漪（面板线另立任务）。

## 实现要点

- 全部改动集中在 `assets/scripts/ui/MainScreenView.ts`，用 Cocos 内置 `tween()`，零新素材。
- 每次起 tween 前 `Tween.stopAllByTarget`，防连点叠加；基准缩放沿用 `pressBaseScale`「只记录一次、回弹完成才删除」的机制，动画中途再按不会把缩小态当基准。
- `build()` 重建时清空 `pressBaseScale`，避免持有已移除节点。
- 精灵原色恒为白（`addSprite` 未改色），松手直接还原为白，不需逐节点记录原色。

## 验证

- `npm run verify` 全绿。
- 用户 Cocos 预览人工点验手感（缩放比例、时长、暗度、涟漪参数均为常量可调）。
