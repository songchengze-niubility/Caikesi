## 一句话

竖屏整体、**核心战斗横向展开**的多人小队自动战斗微信小游戏（Cocos 3.8.8，`F:\Cgame`）。

## 已完成

- 战斗循环：3 人小队（坦克近战/输出远程/治疗），单排一字阵；敌人从右进场、无碰撞群殴。
- 战斗框架：统一 `stats` 属性表 + 唯一减法伤害公式；维度有 暴击/暴伤(加成式)/格挡/闪避/攻速/伤害加成/伤害减免/射程。
- 关卡系统：`enemyTypes` 怪物类型表（丧尸/疾行者/重装）+ `levels` 关卡表（每波可混多种怪），打完该关所有波即通关。
- 表现：色块占位 + 战斗飘字 + 可替换的多层视差滚动背景。
- 工具：游戏内实时调参面板（`ConfigPanel`，临时手感调试）；**Excel 导表管线**（`tools/`，多源驱动 + `npm run config` + 校验 + E2E）。
- **美术资源管线**（`art/` + `resources/art/`）：逻辑键 + 缺图回退色块；ArtManifest 登记 + check-art 校验 + 序列帧动画；接美术不动玩法代码。
- **装备存储系统**（`inventory/`，纯存储层）：背包/仓库/每角色 5 装备栏 + 掉落/转移/穿脱 + 持久化 + 占位 UI。**不影响战斗数值**（下一版接）。
